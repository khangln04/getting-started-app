Build the app's image
1. In the getting-started-app directory, the same location as the package.json file, 
create a file named Dockerfile with the following contents:

# syntax=docker/dockerfile:1

FROM node:24-alpine
WORKDIR /app
COPY . .
RUN npm install --omit=dev
CMD ["node", "src/index.js"]
EXPOSE 3000

2. Build the image using the following commands:
In "cd /path/to/getting-started-app"
cmd 

docker build -t getting-started .

Start an app container
1. Run your container using the docker run command 
and specify the name of the image you just created:

 docker run -d -p 127.0.0.1:3000:3000 getting-started
